## 2. 1ページ概要シート（A4印刷想定）

### PDFノート とは

> 介護・医療・士業など、**大量PDFを扱う事業所のための無料ツール**です。

| 項目 | 内容 |
|------|------|
| 開発元 | lifemate-inc |
| URL | https://lifemate-inc.github.io/pdfnote/ |
| 料金 | 完全無料 |
| インストール | 不要（ブラウザでURLを開くだけ） |
| 対応OS | Windows 10/11、macOS |
| 推奨ブラウザ | Google Chrome / Microsoft Edge |
| データ送信 | **一切なし**（PCの中だけで処理） |

### 主な機能

1. **一括分割保存** — 分割点をクリックで設定 → フォルダを1回選ぶだけで全PDFを保存
2. **氏名候補の自動抽出** — PDFテキスト or OCRで表紙から氏名を抽出、クリックで命名
3. **テキスト追記** — メモ・確認印を任意位置に配置（フォントサイズ/色/太字対応）
4. **ページ回転** — サムネイル上の1クリックで90度回転
5. **自動保存** — 作業中の状態が自動保存され、ブラウザを閉じても復元可能
6. **オフライン動作** — 初回読み込み後はネット不要

### 安全設計

- PDFファイルがパソコンの外に出ない設計（技術的に通信遮断）
- ソースコードを GitHub で全公開（透明性確保）
- 個人情報保護法・各種ガイドラインに準拠した運用が可能

## 3. 3分でわかる使い方

### Step 1: PDFを開く
1. https://lifemate-inc.github.io/pdfnote/ をブラウザで開く
2. PDFをドラッグ&ドロップ（または「クリックしてファイルを選択」）

### Step 2: 分割する
1. 画面右上の「✂ 分割」ボタンをクリック
2. サムネイル間の縦線（ハサミ✂）をクリックして分割点を設定
3. 「次へ →」をクリック
4. 各グループのファイル名を入力（候補ボタンから選んでもOK）
5. 「フォルダを選んで保存」をクリック → フォルダを選ぶ → 完了

### Step 3: テキスト追記する
1. 縦スクロール閲覧画面で右下の「テキストを追加」をクリック
2. テキストを書きたい場所をクリック
3. テキスト・サイズ・色を選んで「追加」
4. 完了後、右上「PDFを保存」で出力

### Step 4: ページ回転する
1. 「ページ一覧」ボタンをクリック（または閲覧画面左サムネ）
2. 各サムネイル右上のホバーボタンで90度回転
3. 「PDFを保存」で確定

## 6. パイロット先選定の優先順位（社内メモ）

| 優先度 | 業種 | 理由 |
|--------|------|------|
| 🔴 最優先 | 介護事業所（小〜中規模） | 100ページ超PDFの典型ユースケース |
| 🟠 高 | 訪問看護ステーション | 利用者ごとの書類分け |
| 🟠 高 | 居宅介護支援事業所 | ケアプラン・帳票PDF |
| 🟡 中 | 社労士・税理士事務所 | 顧客ごとの書類整理 |
| 🟡 中 | 小規模クリニック | 患者書類の仕分け |
| 🟢 検証 | 自治体・教育機関 | 業務多様性検証 |

### 選定基準
- IT担当者がいない or 少ない事業所優先（ターゲット層と一致）
- 個人情報を多く扱う事業（セキュリティ売りが響く）
- すでに業務で困っている事業所（解決体感がある）
