# Smart PDF — 軽量PDF編集ツール

事業所での書類業務を効率化する、シンプルなブラウザ完結型 PDF 編集ツールです。

100ページ超の大量 PDF を「人ごとに分割」「向き修正」「リネーム」「メモ追記」する作業を、**無料で・安全に・誰でも**行えるようにします。

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Status: Phase 0 設計中](https://img.shields.io/badge/Status-Phase%200%20設計中-yellow.svg)](docs/requirements.md)

> **⚠️ 現在は Phase 0（設計フェーズ）です。**
> 正式リリースまでは設計ドキュメントとモックアップのみが整備されています。実装は Phase 1 で進めます。

---

## ✨ 特徴（予定）

- 🔒 **ローカル完結** — PDF はご利用パソコン内で処理され、インターネットを通りません
- 🪶 **軽量・無料** — 必要最低限の機能に絞り、誰でも使えるシンプルさ
- 📑 **大量 PDF に強い** — 100 ページ超の PDF を快適に分割・整理
- ✏️ **メモ書きスタンプ** — PDF の上から文字をスタンプ感覚で配置
- 🔄 **ページ回転** — 横向きページもワンクリックで修正
- 📝 **かんたんリネーム** — 鉛筆アイコンをクリックしてその場で名前変更
- 🖥 **Windows / Mac 両対応** — ブラウザがあれば動きます
- 💾 **自動更新** — インストール不要・常に最新版

---

## 🎯 想定ユーザー

**主要ターゲット:** 介護事業所のスタッフ（IT に不慣れな方でも使えるよう設計）

**拡張ターゲット:** 医療機関 / 士業 / 教育機関 / 自治体 / 中小企業の事務担当者など、100 ページ規模の PDF を日常的に扱うすべての業種・職種

## 📐 動作環境（予定）

| 項目 | 内容 |
|------|------|
| OS | Windows 10 / 11、macOS（最新） |
| ブラウザ | **Google Chrome / Microsoft Edge（最新版）推奨** |
| メモリ | 4GB 以上推奨（大容量 PDF 処理時は 8GB 以上推奨） |
| インストール | **不要**（URL アクセスのみで起動） |

> Safari / Firefox は部分対応（ファイル保存先指定 API 未対応のため、ダウンロード方式にフォールバック）。

## 🔒 セキュリティ設計

PDF の内容は**インターネットに送信されません**。

```
┌─ ご利用パソコン（オフライン完結） ────────────────┐
│   ブラウザ → pdf.js / pdf-lib                       │
│   IndexedDB（作業状態を自動保存）                   │
└────────────────────────────────────────────────────┘
            ↑ インターネット（PDFデータは通りません）
┌─ GitHub Pages（HTML / JS / CSSのみ配信） ─────────┐
│   index.html / app.js / styles.css                  │
└────────────────────────────────────────────────────┘
```

詳細は [セキュリティ設計書](docs/security.md)（Phase 0 後半で作成予定）をご覧ください。

## 🚫 やらないこと（スコープ外）

開発をシンプルに保つため、以下は明示的にスコープ外とします:

- PDF 内既存テキストの編集（スタンプによる上書きのみ）
- OCR（画像 PDF の文字認識）
- フォーム入力機能
- パスワード保護 PDF の解除
- 電子署名（→ 姉妹プロジェクト [hpki-signer](https://github.com/lifemate-inc/hpki-signer) をご利用ください）
- 複数 PDF の結合
- クラウド保存・同期
- スマートフォン対応
